# Flutter Projects

A collection of Flutter mobile apps built with Dart during learning and development.

## Apps

| Folder | App | Description |
|---|---|---|
| `ecommerce_project` | KFC Clone UI | Food ordering app UI inspired by KFC — home, menu, cart, and checkout pages |
| `first-app` | Card Info App | Displays personal info cards with tap-to-copy and link-opening functionality |
| `weather_app` | Weather App | Shows real-time weather using geolocation and a weather API |
| `news_app` | News App | Fetches and displays live news articles by category |
| `tasbeeh_app` | Tasbeeh Counter | Islamic prayer counter with settings for vibration and sound |
| `converter_app` | Currency Converter | Converts between Pakistani Rupees and US Dollars |
| `login_form` | Login UI | Clean login screen UI with form validation |
| `quirky_quills` | Bookstore App | Bookstore UI — work in progress |

## Built With
- Flutter
- Dart
- Supabase
- Provider (state management)
